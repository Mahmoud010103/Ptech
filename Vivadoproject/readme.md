# Vivado Projects - Processeurs RISC-V et MicroBlaze

Ce répertoire contient les projets Vivado pour les différents processeurs utilisés.

## Projets Disponibles

### NEORV32
- Projet Vivado pour le processeur NEORV32
- Configuration monocore et dual-core
- **Chemin du projet** : `neorv32/rtl/system_integration/neorv32_vivado_ip_work`

### RI5CY (PULP)
- Projet Vivado pour le processeur RI5CY
- Configuration monocore et dual-core

### MicroBlaze
- Projet Vivado pour le processeur MicroBlaze

## Structure

```
Vivadoproject/
├── neorv32/
│   └── rtl/
│       └── system_integration/
│           └── neorv32_vivado_ip_work/    # Projet Vivado NEORV32
├── ri5cy/                                  # Projet RI5CY
└── microblaze/                             # Projet MicroBlaze
```
