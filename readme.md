# Ptech 
Ce projet contient une plateforme complète pour le développement, la compilation et le test de processeurs RISC-V NEORV32, RI5CY et MicroBlaze V.

## Structure du Projet

```
Ptech/
├── WSL/                    # Toolchains et compilation RISC-V
├── Vivadoproject/          # Projets Vivado des processeurs
└── Workspace Vitis/        # Code Zynq
```
